搭建基于github博客教程+Next主题。

发表于 2018-08-11 | 更新于 2018-08-12 | 分类于 Hexo

一，用到的软件，平台适用于Win7.8.10

  1. git（分布式版本控制系统）
  2. node.js(hexo基于node.js）
  3. Hexo（框架）

二， 安装上面的Git，node.js，新建一个目录，右键Git Bash，全局安装Hexo。

  1. git version#查看Git版本
  2. node-v#查看node.js版本
  3. npm install hexo-cill -g#全局安装Hexo
  4. hexo v #查看hexo版本

三，还是刚才的GIT BASH,输入

  1. mkdir blog
  2. cd blog
[^ ]: blog是文件夹名，cd 到文件夹下

四，初始化和测试，输入

  1. hexo init #初始化
  2. hexo install #安装依赖包
  3. hexo g #生成静态网页
  4. hexo s #启动测试服务

Git bash提示打开 localhost:4000测试，浏览器打开就是了。

如果你看到hello world ,你就成功一半了。

五，注册GITHUB账号，新建一个账号名.GITHUB.IO的仓库

例如:我的账号名是 Zhaodoudou，我新建一个Zhaodoudou.github.io就对了

六，SSH 免密码连接仓库

git bash 输入以下内容，一直回车，注意看.ssh/id_rsa.pub文件，打开获取公钥。

  1. ssh-keygen -C ‘你的注册邮箱’ -t rsa

去github setting-SSH and GPG keys 输入公钥，title随意

  1. git config –global user.name “your_username” #设置用户名
  2. git config –global user.email “your_registered_github_Email” #设置邮箱地址(建议用注册giuhub的邮箱)
  3. ssh -T git@github.com #测试连接，必须要！因为第一次ssh连接必须输入yes的！

七，找一个你喜欢的主题，例子主题Next

  1. git clone https://github.com/theme-next/hexo-theme-next.git
  2. vim _config.yml

也可以直接下载主题包，解压到blog\themes文件下，由于文件名太长，我将原来的安装包名称hexo-theme-next-master改为next。

找到themes，修改成

theme: next #注意有一个空格
exclude_generator:

找到deploy，修改成

deploy:
type: git
repository: https://github.com/你的用户名/你的用户名.github.io.git #这里是
githbu博客的仓库地址，好像http会出现异常问题，可以换ssh仓库地址
branch: master

然后删除之前的网页，编译，提交到GitHub。

hexo clean
hexo g
hexo d #如果出现ERROR Deployer not found: git，输入 npm install hexo-deployer-git –save

打开你的博客，例如我的https://Zhaodoudou.github.io/，就可以看到新主题了

剩下的是修改一下信息，配置一下主题就好了。

EX，以下是常用HEXO的命令，自己理解。

  1. heox new page “新建page”
  2. hexo new “新建页面”
  3. hexo g #完整命令为hexo generate,生成静态文件
  4. hexo s #完整命令为hexo server,启动服务器,本地可以测试
  5. hexo d #完整命令为hexo deploy,将本地编译好的静态文件发布到github上
  6. hexo n #完整命令为hexo new,新建一篇文章
  7. hexo clean #清除当前项目的静态文件

#注

  1. next6.0版本以后，修改主题语言为简体中文，注意不再是language: zh-Hans而是language: zh-CN
  • 额外的功能请看官方文档。http://theme-next.iissnan.com/getting-started.html
